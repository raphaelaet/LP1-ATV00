# LP1-ATV00 - Termômetro

## Como compilar e rodar
1. Tenha um compilador C instalado na sua máquina (ex.: **gcc**).
2. Clone o repositório usando o comando `git clone https://github.com/raphaelaet/LP1-ATV00.git` ou baixe os arquivos em ZIP.
3. Abra o terminal na pasta do projeto.
4. Compile o código da questão desejada substituindo o "X" pelo número da questão no comando `gcc qX.c -o qX -g -W`:
5. Execute o programa gerado:
   - No Windows: `.\qX.exe`
   - No Linux: `./qX`
